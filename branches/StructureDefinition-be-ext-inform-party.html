Extension: BeInformParty

Extra parties that need to be informed of the outcome, besides the prescriber.

The official URL for this extension is: 

https://www.ehealth.fgov.be/standards/fhir/StructureDefinition/be-ext-inform-party

Status: draft
Extension maintained by: HL7 Belgium

Context of Use

This extension may be used on the following element(s):

  • {"type"=>"Resource"}

Usage info

Usage:

  • Use this Extension: BeLabPrescription, BeReferralPrescriptionDiagnosticImaging, BeReferralPrescriptionNursingBladderCare, BeReferralPrescriptionNursingCompressionTherapy, BeReferralPrescriptionNursingDigestiveSystemCare, BeReferralPrescriptionNursingMedication, BeReferralPrescriptionNursing, BeReferralPrescription
