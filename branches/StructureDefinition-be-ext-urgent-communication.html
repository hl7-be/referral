Extension: BeUrgentCommunication

Official URL: https://www.ehealth.fgov.be/standards/fhir/referral/StructureDefinition/be-ext-urgent-communication Version: 0.2.0
Draft as of 2021-07-15 Responsible: HL7 Belgium Realm: Computable Name: BeUrgentCommunication

When there is a need to urgently communicate results of an order.

Context of Use

This extension may be used on the following element(s):

  • ServiceRequest

Usage info

Usage:

  • Use this Extension: BeLabPrescription
  • Examples for this Extension: ServiceRequest/laboratory-order-example1
