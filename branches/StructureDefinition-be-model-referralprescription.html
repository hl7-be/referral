StructureDefinition: BeReferralPrescription

Logical model describing the model for the prescription referral. This draft is used in the HL7 Belgian workgroup for referral.

The official URL for this profile is: 

https://www.ehealth.fgov.be/standards/fhir/StructureDefinition/be-model-referralprescription
