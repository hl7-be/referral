Resource Profile: BeReferralPrescriptionNursingMedication

Defining URL:https://www.ehealth.fgov.be/standards/fhir/StructureDefinition/be-referralprescription-nursing-medication
Version:0.2.0
Name:BeReferralPrescriptionNursingMedication
Status:Draft as of 2021-10-15T15:30:39+00:00
Definition:

The nursing profile specialized for medication. Note this profile is based on the FHIR resource MedicationRequest. Dosage is expected initially to be given in the corresponding text and optional site fields (as per the NIHDI requirements)

Publisher:HL7 Belgium
Source Resource:XML / JSON / Turtle
